# 스마트 팩토리 전력 소비 예측 및 최적 부하 스케줄링

- 제4회 BDAI 채용 연계 데이터 분석 공모전
- 주최: BDAI x AINATION
- 기간: 2025.12 – 2026.01

---

## 1. 프로젝트 개요

스마트 팩토리의 전력 소비량을 예측하고, 예측값을 기반으로 두 가지 문제를 해결한다.

1. 이상 전력 패턴 사전 탐지: 예측된 정상 범위를 벗어나는 구간을 이상 신호로 판단하여 설비 이상을 사전 감지
2. TOU 기반 전력 스케줄링: 예측 전력량을 바탕으로 계시별 요금제 구조에 맞게 머신의 부하를 재배치하여 전력 비용 최소화

---

## 2. 문제 정의

| 문제 | 내용 |
|------|------|
| 이상 설비 사후 대응 | 스마트 팩토리는 자동 제어 구조로 인적 개입이 최소화되어 설비 이상 발생 시 즉각 조치가 어려움 |
| 전력 요금 낭비 | 고부하 시간대(158.2원/kWh)에 전력 소비가 집중되어 불필요한 비용 발생 |

TOU 요금 체계:

| 구분 | 단가(원/kWh) | 시간대 |
|------|-------------|--------|
| 경부하 | 105.6 | 22:00~08:00 |
| 중간부하 | 127.9 | 08:00~11:00, 12:00~13:00, 18:00~22:00 |
| 고부하 | 158.2 | 11:00~12:00, 13:00~18:00 |

## 4. 전처리 및 피처 엔지니어링

- `accumActiveEnergy` 등 중복 변수 제거
- 설비 가동 여부(`operation`) 변수 적용: 미가동 시 예측값 0 처리
- 외부 데이터(기상청) 검토 → 전력과의 상관계수 0에 수렴하여 제외

시계열 피처 생성:

| 피처 | 설명 | Feature Importance |
|------|------|--------------------|
| rolling_24 | 최근 24시간 이동평균 | 55.5% |
| lag_24 | 전일 동시간 전력 | 14.3% |
| lag_1 | 직전 1시간 전력 | 12.5% |
| hour | 시간대 | 12.5% |
| dayofweek | 요일 | 3.0% |

## 6. 솔루션

### 6-1. 이상 전력 패턴 탐지

- 위험 구간: 전력 사용량 분포의 μ ± 3σ 초과 구간
- 이상 감지 시 설비명, 전력 사용량 기준치 초과 여부, 전류 불평형률 알림 
- 전류 불평형 계산으로 과부하 발생 상(R/S/T) 자동 추적

### 6-2. TOU 기반 선형계획법 전력 스케줄링

목적함수:

```
Minimize Z = Σ(Cₜ × Pₜ) + α × P_peak + β × Σ|Pₜ - Pₜ₋₁|
```

제약조건:
1. 일일 총 전력 소비량 보존
2. 시간당 허용 최대 변동폭 이내
3. 월간 최대 수요 전력 이하
4. 설비 가동 최소/최대 한계값 준수

절감 효과 (5월 기준):
- 기존 예측 요금: 34억 9,920만원
- 최적 스케줄 요금: 34억 9,880만원
- 월간 절감: 약 40만원 / 연간 절감: 약 480만원
